# Contributing Guidelines

Thank you for considering contributing to **claude-times**.

## Code Standards

-   Python 3.8+ compatibility
-   Clear, readable, and maintainable code
-   Proper docstrings and inline documentation
-   Follow PEP8 guidelines

## Pull Request Process

1.  Fork the repository.
2.  Create a feature branch.
3.  Submit a Pull Request with a clear description.
4.  Ensure changes maintain AGPL-3.0 license compatibility.

## Licensing of Contributions

By submitting a contribution, you agree that your contribution will be
licensed under AGPL-3.0-or-later.
